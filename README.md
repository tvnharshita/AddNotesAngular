# AddNotesAngular
A sample project to add notes with time and delete the notes

Download the app from github into your local folder.
To run the app:
1. If you donot have node in you machine, download the latest version of it online.
2. Install Angular Cli with the command "npm install -g @angular/cli".
3. Open command prompt and go to your downloaded folder.
4. Run the command "ng serve --open".
